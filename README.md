# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

### Production build

To build the web version for production:

```bash
npm install
npm run build
```

To export static web assets:

```bash
npm run export:web
```

For native Android/iOS production builds, use EAS:

```bash
npm install -g eas-cli
npm run eas:build:android
npm run eas:build:ios
```

When you're ready, you'll find the web build output in `web-build/` or the exported web assets in `web-export/`, depending on Expo.

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.
